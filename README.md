# GymPulse---Gym-Management-System
"A high-performance Full-Stack Gym Management System built with Java 24 and Spring Boot 4.0.1. Features automated attendance tracking, slot-based enrollment limits, and real-time business intelligence reporting with Oracle 11g/XE integration."

Slot Constraints: Custom logic to enforce a 10-member limit per shift for gym safety and capacity management.

BI Dashboard: Real-time revenue calculation and member distribution analytics using Chart.js.

Optimized Persistence: Leveraging JPA Sequence Generators and HikariCP for robust Oracle DB interactions.

Clean API Design: RESTful endpoints with recursive-safe JSON serialization (using @JsonIgnoreProperties).

Security: Integrated Spring Security for protected administrative access.
